# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In Short

Be respectful and constructive. We're here to make Windows 11 better, not to argue.

## Scope

This applies to all project spaces: GitHub issues, pull requests, and discussions. It also applies when representing the project in public spaces.

## Standards

**Do:** give constructive feedback, accept criticism gracefully, help newcomers, credit sources for registry tweaks.

**Don't:** personal attacks, trolling, spam, publishing others' private info.

## Reporting

Report privately via [GitHub Security Advisories](https://github.com/emylfy/winrift/security/advisories/new). All reports will be reviewed confidentially.
